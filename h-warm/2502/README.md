# Warm2502

Warm2502 is the home-energy Digital Twin project.

## Source of truth
GitHub is the primary source of truth for architecture, decisions, datasets, models, workflows, backlogs and agent behaviour.

Start here:

1. `project-register.md`
2. `architecture/ADR-2026-06-06-warm2502-digital-twin.md`

## Goal
Evolve Warm2502 from chat/files/analyses into a reproducible Digital Twin that can process current data, preserve historical energy data, run simulations, compare scenarios, evaluate investments and reproduce results.
