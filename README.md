# Output archive

Duurzame opslag voor gegenereerde rapporten en bestanden uit ChatGPT.

## AI World Journal

Dagelijkse AI-rapporten worden opgeslagen onder:

```text
ai-world-journal/YYYY/MM/yymmdd-hh:mm-ai-deep-dive.html
```

Voorbeeld:

```text
ai-world-journal/2026/05/260518-20:00-ai-deep-dive.html
```

Doel:

- blijvende links in GitHub;
- geen tijdelijke `sandbox:/mnt/data`-links meer;
- rapporten leesbaar als HTML;
- opbouwend archief voor AI-signalen, trends, risico's en handelingsperspectief.
